# Contributing

Thank you for your interest in improving this plugin. Contributions are welcome and appreciated.

## What to contribute

This is a knowledge plugin — the main content lives in `skills/pikud-haoref-alerts/`. The most valuable contributions are:

- **Corrections** — Fix inaccurate endpoint details, outdated library versions, or wrong instructions
- **New content** — Document additional community libraries, integration patterns, or deployment strategies
- **Translations** — Improve Hebrew/English/Arabic/Russian location name coverage

## How to contribute

1. Fork this repository
2. Create a branch (`git checkout -b fix/description`)
3. Make your changes
4. Submit a pull request with a clear description of what changed and why

## Reporting issues

If you find outdated or inaccurate information, please [open an issue](https://github.com/yaniv-golan/pikud-haoref-alerts/issues/new). Include:

- What's wrong or outdated
- What the correct information should be (with a source if possible)

## Guidelines

- Keep content factual and sourced where possible
- The official oref.org.il endpoints are reverse-engineered and undocumented — note when something is observed behavior vs. guaranteed
- Test any code examples you add
- This plugin covers Israeli civil defense alerts only — other countries' alert systems are out of scope
